# 🚀 Modern Glassmorphism Developer Portfolio

A visually stunning, responsive, and futuristic portfolio website designed for Computer Science undergraduates and developers. Built with a dark theme, deep navy-to-purple gradients, glassmorphism UI cards, and soft neon glow accents.

## ✨ Features

- **Dynamic Hero Section**: Features an interactive typing animation showcasing different roles and interest fields, plus floating parallax gradient backgrounds.
- **Skills & Growth Journey**: Categorized skills with interactive, scroll-triggered progress bars and live "Currently Improving" badges to show a growth mindset.
- **Interactive Projects Grid**: Grid layout of projects with technology tags, source code/demo links, and a subtle 3D hover tilt effect.
- **Interactive Timelines**: Timeline-based sections for highlighting professional experience and academic education.
- **Coding Profiles Integration**: Showcases problem-solving metrics across platforms (LeetCode, CodeChef, GeeksforGeeks) with animated number counters that trigger on scroll.
- **Certifications & Badges**: Clean card layouts with hover glow effects and links to verification certificates.
- **Smooth Navigation**: Custom scroll progress indicator, smooth-scroll links, and a responsive mobile drawer navigation menu.

## 🛠️ Tech Stack

- **Markup**: HTML5 (Semantic Structure)
- **Styling**: CSS3 (Vanilla CSS variables, Flexbox/Grid, Custom animations)
- **Interactions**: Vanilla JavaScript (no external libraries like jQuery)
  - *Intersection Observer API* for scroll-triggered animations.
  - *requestAnimationFrame* for smooth performance-friendly counters.

## 🚀 Getting Started

### Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
   ```
2. Open `index.html` in any web browser to view the site.

### Deployment (Vercel)
The easiest way to host this site is via Vercel:
1. Push your repository to GitHub.
2. Link your GitHub account to [Vercel](https://vercel.com).
3. Import this repository and click **Deploy**.
